How to Use Número Español


Welcome to Número Español, a simple and fun way to learn Spanish numbers! Follow these steps:

  1. Use the range slider to set the desired number range.
  2. A random number will be generated within the selected range.
  3. Type the correct Spanish name for the displayed number in the input field.
  4. Click the "Submit" button to check your answer.
  5. The app will display the correct Spanish name, your answer, and indicate if it's correct or not.
  6. Click the "Refresh" button to get a new random number within the range.


Enjoy learning Spanish numbers with Número Español!
